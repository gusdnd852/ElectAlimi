 * Fan - 5V
 * Neopixel - Adafruit
 

<br>
<br>

## H/W Detail
Android의 베이스가 firebase로 되어 있어 해당부분과의 원할한 연동을 위하여 NodeMCU를 사용하였다.
ESP8266wifi.h 와 FirebaseArduino.h 를 이용하여 연동함.
전력상태를 확인하는 상태를 가정하기 위하여 fan 과 Neopixel를 이용하여 프로토타이핑을 하였다.
